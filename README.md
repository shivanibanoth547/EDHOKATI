# Simple Todo App

A clean and simple todo application built with HTML, CSS, and JavaScript.

## Features

- ✅ Add new tasks
- ✅ Mark tasks as complete
- ✅ Delete tasks
- ✅ Data persistence using localStorage
- ✅ Clean, responsive design

## How to Use

1. Clone this repository
2. Open `index.html` in your web browser
3. Start adding your tasks!

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
